# ASCII Art 图片转换插件
这是一个测试
这是一个为 AstrBot 开发的插件，可以将用户发送的图片转换为 ASCII 艺术形式。
p:当前支持了用中文转换

## 功能

- 将图片转换为 ASCII （或中文）字符画
- 自动调整输出尺寸

## 使用方法

1. 引用该图片并发送命令 `/ascii`
2. 等待插件处理并返回转换后的 ASCII 图片

3. 你可以在配置面板自定义一些配置，支持使用时指定部分参数

## 注意事项

- 插件会在处理完成后自动清理临时文件（如果未知原因没成功的话只能手动清理一下了）
- 转换质量取决于原图的清晰度和尺寸（当然你也可以手动指定出图的宽度）
- 大尺寸图片可能需要更长时间处理（特别是主机性能比较差时）
- 非常不建议宽度参数调整到1000或以上，建议只用插件自身处理方法
- 只在QQ上测试过（napcat）或docker


## 插件使用 Pillow 库进行图像处理，通过灰度化和字符映射的方式生成 ASCII 艺术，并最终将结果保存为图片发送回用户。
